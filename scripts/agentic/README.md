# Agentic Harness Scripts

Deterministic agentic flow harness entry points.

- `agentic_harness_bootstrap.sh`: Create a venv and install agentic harness dependencies.
- `agentic_harness.sh`: Run the deterministic agentic flow harness (supports scenario/driver filtering).

Examples:

- `scripts/agentic/agentic_harness_bootstrap.sh`
- `scripts/agentic/agentic_harness.sh --mode=deterministic`
