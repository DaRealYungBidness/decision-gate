# Minimal Example

## Overview
Runs a minimal Decision Gate scenario using in-memory adapters. Demonstrates
`scenario_next`, `scenario_status`, and runpack generation.

## Run
```bash
cargo run -p decision-gate-example-minimal
```

## Notes
- Uses in-memory providers and dispatchers.
- Intended for quick sanity checks, not production.
