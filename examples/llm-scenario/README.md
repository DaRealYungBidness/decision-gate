# LLM Scenario Example

## Overview
Demonstrates a disclosure flow that dispatches a prompt payload to a callback
sink, simulating an LLM integration and model submission capture.

## Run
```bash
cargo run -p decision-gate-example-llm-scenario
```

## Notes
- Uses broker callback sinks for simulated LLM handling.
- Intended for integration modeling, not production.
